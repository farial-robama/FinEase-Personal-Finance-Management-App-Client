# FinEase - Personal Finance Management App
FinEase is a modern personal finance management web application that helps user track income, expenses, and savings goals with real-time insights. It includes secure authentication, data visualization, transaction history, CRUD operations, and personalized financial reporting.

**Live Website:** https://finease-client.web.app/

## Features

**User Authentication (Firebase)** </br>
- Email/Password registration & login </br>
- Google login </br>
- Password reset option </br>
- Secure logout functionality

**Protected Routes** </br>
- Only logged-in users can access game details and certain pages

**Full CRUD operations** </br>
- For managing transactions.

**Reports page with charts** </br>
- Income vs expense, category insights

**Monthly filtering & sorting** </br>
- (Date, Amount)

**User Profile** </br>
- Update display name & photo URL </br>
- View user details by clicking on the user image in the navbar

**Home Page** 

**Light/Dark Mode** 

**Responsive Design** 
- Fully mobile and tablet friendly

**UI Enhancements** 
- Animated banner (Framer Motion & Daisy UI Carousel) 
- Toast notification for login/logout 
- Loading spinners and error handling

**Routing & Navigation** 
- Dynamic routes using React Router 
- 404 Not Found page 
- Navbar & footer across all pages

## Technologies Used 
### **Frontend**
 - React.js, Vite, JavaScript(ES6+), Tailwind CSS, DaisyUI 
 - React Router 
 - Framer Motion 
 - Firebase Auth 
 - Chart.Js
 - React Hot Toast/SweetAlert2 

### **Backend**
 - Node.js 
 - Express.js 
 - MongoDB 
 - CORS, dotenv 
 - Vercel Deployment
